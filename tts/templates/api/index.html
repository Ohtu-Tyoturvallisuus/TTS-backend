Welcome to the HazardHunt API

Below is a list of available API endpoints and their links:

  • Worksites /api/worksites/
    • List all worksites - GET ../api/worksites/
    • Create a new worksite - POST ../api/worksites/
    • Retrieve, update, or delete a specific worksite - GET ../api/worksites/<worksite_id>/
  • Surveys /api/surveys/
    • List surveys for a specific worksite - GET ../api/worksites/<worksite_id>/surveys/
    • Create a survey for a specific worksite - POST ../api/worksites/<worksite_id>/surveys/
    • List all surveys - GET ../api/surveys/
    • Create a new survey - POST ../api/surveys/
    • Retrieve, update, or delete a specific survey - GET ../api/surveys/<survey_id>/
  • Authentication
    • Sign in a user - POST /api/signin/
