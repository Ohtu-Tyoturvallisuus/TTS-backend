Welcome to the HazardHunt API

Below is a list of available API endpoints and their links:

  • Projects /api/projects/
    • List all projects - GET ../api/projects/
    • Create a new projects - POST ../api/projects/
    • Retrieve, update, or delete a specific projects - GET ../api/projects/<projects_id>/
  • Surveys /api/surveys/
    • List surveys for a specific projects - GET ../api/projects/<projects_id>/surveys/
    • Create a survey for a specific projects - POST ../api/projects/<projects_id>/surveys/
    • List all surveys - GET ../api/surveys/
    • Create a new survey - POST ../api/surveys/
    • Retrieve, update, or delete a specific survey - GET ../api/surveys/<survey_id>/
  • Authentication
    • Sign in a user - POST /api/signin/
